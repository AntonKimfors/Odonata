
Instructions for starting application:



1. Download and install Android Studio: https://developer.android.com/studio/index.html

2. Import project and proceed to step 3 or 4.

3. If you wish to test the application on a physical android device, see following tutorial: https://developer.android.com/studio/run/device.html if not, proceed to step 4.

4. If you wish to test the application on a virtual device, see following tutorial: https://developer.android.com/studio/run/emulator.html
